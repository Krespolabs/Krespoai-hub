# KrespoAI Hub 🤖
Nigeria-first AI SaaS platform for creators and entrepreneurs.

KrespoAI Hub is a React-based web platform providing AI-powered tools, prompt packs, and automation systems built specifically for the African creator economy.

## Tech Stack
- Framework: React + Vite
- Language: JavaScript JSX
- Deployment: Vercel
- Live: https://krespoai-hub.vercel.app

## Project Structure
- App.jsx — Root application component
- main.jsx — Entry point
- index.html — HTML shell
- vite.config.js — Vite configuration
- vercel.json — Deployment config

## Features
- AI tools and prompt systems for creators
- Automation workflows for Nigerian entrepreneurs
- Clean fast SPA architecture via Vite
- Production-deployed on Vercel

## Install and Run
- git clone https://github.com/Krespolabs/Krespoai-hub.git
- cd Krespoai-hub
- npm install
- npm run dev

## Part of Krespo Labs
KrespoAI Hub is one of several products under the Krespo Labs brand — building AI infrastructure for the African creator and developer ecosystem.
